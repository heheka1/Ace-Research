# Asexual History
This is a project repo dedicated to creating a 
user-friendly database dedicated to various historical 
documents that are related to asexual history. For many years asexual history has been erased and ignored. One aim of this database is to 
           combat asexual erasure and educate others about asexual history. 
           As of now, most documents will be consisted of ones that 
           can be scraped or are able to be processed digitally.
